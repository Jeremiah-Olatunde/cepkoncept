### Project Scripts

*Clean ./dist and ./.parcel-cache folders*
```shell
npm run clean
```

*Start parcel watcher*
```shell
npm run start
```

*Start live server*
```shell
npm run serve
```

*Build final distribution*
```build
npm run build
```
